## 1.0 版本! 🎉🎉🎉

Next-Admin 1.0.0

## 特点

- Next14.0 + antd5.0
- 支持国际化
- 支持主题切换
- 内置数据可视化报表
- 开箱即用的业务页面模板
- 支持自定义拖拽看板
- 集成办公白板
- Next全栈最佳实践

## 使用

### 安装

```shell
$ git clone git@github.com:MrXujiang/next-admin.git
```

安装依赖:

```shell
$ cd next-admin && pnpm install
```
